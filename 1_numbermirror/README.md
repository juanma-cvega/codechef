# Number mirror

## Problem

Write a program that takes a number N as the input, and prints it to the output.

## Input Format

The only line contains a single integer.

## Output Format

Output the answer in a single line.

## Constraints

0 ≤ N ≤ 1

## Sample 1

| Input | Output |
| ----- | ------ |
| 123 | 123 |
| 15 | 15
